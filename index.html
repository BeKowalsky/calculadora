Calculadora para operações básicas

Como usar a calculadora:

➥IMPORTANTE: A calculadora aceita SOMENTE 12 dígitos

➔ Para ligar a calculadora, basta apertar em "ON/C", Assim que mudar de OFF para ON a calculadora está pronta para ser usada.

➔ Se deseja salvar um valor para ser usado posteriormente, aperte "M+"".

➔ Se desejar ver o valor armazenado aperte "MRC", caso você aperte "MRC" duas vezes, a calculadora irá apagar o valor armazenado.

➔ Se desejar subtrair o valor atual da calculadora com o valor armazenado, aperte "M-" que a operação irá acontecer. O valor da subtração será armazenado na calculadora, não é preciso apertar "M+" para armazena-lo.

➔ Se quiser usar cálculos com vírgula, é preciso apenas apertar no "●", é importante saber que esse botão serve apenas para colocar vírgula no número, se quiser fazer cálculos com valores em mil, milhão, não deve colocar o ponto.

➔ Se você acabou de realizar um cálculo e quer fazer um novo, só clicar em "CE" que a calculadora apaga os valores.
